Discipline

The discipline domain represents discipline incidents and the action taken to rectify the incident.

Discipline Model Entities

Name Description
PersonDisciplineAction Defines the actions assigned to an offender.
Person Defines all the individuals associated with an education organization.
DisciplineIncidentCode This represents the additional attributes required for the Behavior descriptor.
PersonDisciplineIncidentAssociation This association indicates those persons who were victims, offenders and witnesses for a discipline incident.
