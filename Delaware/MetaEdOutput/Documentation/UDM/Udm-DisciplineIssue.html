Discipline Issue

This domain represents discipline issues captured in the Teacher Access Center.

Discipline Issue Model Entities

Name Description
DisciplineIssue This represents discipline issues entered through the teacher access center.
DisciplineIssueRefer This represents the referral information on discipline issues that were referred.
DisciplineIssueAction This represents the action associated with a discipline issue.
DisciplineIssueStudent This represents the student associated with the discipline issue.
Related This represents related issues.
